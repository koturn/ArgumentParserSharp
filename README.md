ArgumentParserSharp

## Build

```shell
> msbuild /nologo /m /t:restore /p:Configuration=Release;Platform="Any CPU" ArgumentParserSharp.sln
> msbuild /nologo /m /p:Configuration=Release;Platform="Any CPU" ArgumentParserSharp.sln
```
